## そもそもSSG?ISR?SSRって何？
レンダリング方法です！めっちゃさらっと紹介しているので、時間がある方はこちらの記事を確認してからお読みいただくと理解が深まると思います。
https://zenn.dev/akino/articles/78479998efef55

### SSG （Stactic Site Generation）
ビルド時に構築した静的なサイトを利用

### ISR (Incremental Static Regeneration)
静的サイトの一部を一部動的に更新する

### SSR (Server Side Rendering)
毎回サーバーにリクエスト

## さっそく実装
### 1. microCMSセットアップ

こちらからmicrocmsのダッシュボードに登録し、新規プロジェクトを作成してください。
プロジェクトの作成後、コンテンツを作成しましょう。
今回は、基本の「ブログテンプレート」を使用しました。

エディタに戻ってnextのプロジェクトを作成します。
```
npx create-next-app@latest next-ssr-blog
```

お忘れなく
```
npm install microcms-js-sdk
```

next-ssr-blog/
├── config.ts // 共通のデザイン部品（ここを1つにする）
├── .env      // MicroCMSの環境変数入れる


```.env
MICROCMS_SERVICE_DOMAIN=next-ssr-<あなたのドメイン名をセット>
MICROCMS_API_KEY=<MicroCMSのAPIキーをセット>
```

microcmsの設定ファイルを作成します。
```microcms.ts
import {createClient} from "microcms-js-sdk";

export const client = createClient({
    serviceDomain: process.env.MICROCMS_SERVICE_DOMAIN || "",
    apiKey: process.env.MICROCMS_API_KEY || "",
});

export type Article = {
    id: string;
    createdAt: string;
    updatedAt: string;
    publishedAt: string;
    revisedAt:string;
    title: string;
    content: string;
};

export type ArticleResponse = {
    contents: Article[];
    fetchedAt: string;
}

// キャッシュ設定を引数で受け取れるようにする
export const getArticles = async (requestInit?: RequestInit): Promise<ArticleResponse> => {
    const data = await client.getList<Article>({
        endpoint: "blogs",
        customRequestInit: requestInit, // ここにキャッシュ設定を流し込む
    });

    return {
      contents: data.contents,
      fetchedAt: new Date().toLocaleString("ja-JP", { timeZone: "Asia/Tokyo" }),
    };
};
```

### 2. app/以下に検証用ページの作成

app以下はこのようなディレクトリ構成で行います。
```
app/
├── components/
│   └── ArticleList.tsx   // 共通のデザイン部品（ここを1つにする）
├── libs/
│   └── microcms.ts       // 共通のデータ取得ロジック
├── ssg/
│   └── page.tsx          // SSGデモ用ページ
├── isr/
│   └── page.tsx          // ISRデモ用ページ
└── ssr/
    └── page.tsx          // SSRデモ用ページ
```

ブログ表示用に共通のコンポーネントを用意します。（UIはclaudeくんが作ってくれました）
```ArticleList.tsx
import { ArticleResponse } from "../libs/microcms";

type Props = {
  data: ArticleResponse;
  mode: "SSG" | "ISR" | "SSR";
  description: string;
};

export default function ArticleList({ data, mode, description }: Props) {
  return (
    <div className="max-w-2xl mx-auto p-6">
      <div className="mb-8 border-l-4 border-blue-500 pl-4">
        <h1 className="text-3xl font-bold mb-2">{mode} 検証ページ</h1>
        <p className="text-gray-600 mb-4">{description}</p>

        {/* ここが検証のキモ：データ取得時刻 */}
        <div className="bg-yellow-100 text-red-700 p-4 rounded font-mono font-bold text-lg border border-yellow-300">
          データ取得時刻: {data.fetchedAt}
        </div>
      </div>

      <div className="space-y-8">
        {data.contents.map((article) => (
          <article key={article.id} className="border rounded-lg p-6 shadow-sm">
            <h2 className="text-2xl font-bold mb-4">{article.title}</h2>

            {/* リッチエディタの内容を表示 */}
            <div
              className="prose max-w-none"
              dangerouslySetInnerHTML={{ __html: article.content }}
            />

            <p className="text-sm text-gray-400 mt-4 text-right">
              更新日: {new Date(article.updatedAt).toLocaleDateString()}
            </p>
          </article>
        ))}
      </div>
    </div>
  );
}
```

```isr/page.tsx
import { getArticles } from "../libs/microcms";
import ArticleList from "../components/ArticleList";

export default async function IsrPage() {
  const data = await getArticles({
    next: { revalidate: 10},
  });

  return (
    <ArticleList
      data={data}
      mode="ISR"
      description="10秒ごとに再生成のチャンスがあります。連続でリロードして挙動を確認してください。"
    />
  );
}
```

```ssr/page.tsx
import { getArticles } from "../libs/microcms";
import ArticleList from "../components/ArticleList";

export const dynamic = "force-dynamic"; // 動的ページなので、ビルド時に無視してもらう

export default async function SsrPage() {
  const data = await getArticles({
    cache: "no-store", // キャッシュしない
  });

  return (
    <ArticleList
      data={data}
      mode="SSR"
      description="リクエストのたびにサーバーで生成されます。リロードするたびに時刻が変わります。"
    />
  );
}
```

```ssg/page.tsx
import {getArticles} from "../libs/microcms";
import ArticleList from "../components/ArticleList";

export default async function SsgPage() {
    const data = await getArticles();

    return (
        <ArticleList
        data={data}
        mode="SSG"
        description="ビルド時に生成されます。リロードしても時刻は変わりません。"
        />
    );
}
```
